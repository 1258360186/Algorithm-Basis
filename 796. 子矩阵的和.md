##  796. 子矩阵的和

### 题目

输入一个n行m列的整数矩阵，再输入q个询问，每个询问包含四个整数$x_1, y_1, x_2, y_2$，表示一个子矩阵的左上角坐标和右下角坐标。

对于每个询问输出子矩阵中所有数的和。

### 输入格式

第一行包含三个整数n，m，q。

接下来n行，每行包含m个整数，表示整数矩阵。

接下来q行，每行包含四个整数$x_1, y_1, x_2, y_2$，表示一组询问。

### 输出格式

共q行，每行输出一个询问的结果。

### 数据范围

$1≤n,m≤1000$,
$1≤q≤200000$,
$1≤x_1≤x_2≤n$,
$1≤y_1≤y_2≤m$,
$−1000≤矩阵内元素的值≤1000$

### 输入样例

```
3 4 3
1 7 2 4
3 6 2 8
2 1 2 3
1 1 2 2
2 1 3 4
1 3 3 4
```

### 输出样例

```
17
27
21
```

### AC代码

```c++
#include <iostream>
#include <algorithm>

using namespace std;

int n,m,q;
int g[1010][1010];

int main()
{
    cin >> n >> m >> q;
    for(int i=1;i<=n;i++)
        for(int j=1;j<=m;j++)
        {
            cin >> g[i][j];
            g[i][j] += g[i-1][j]+g[i][j-1]-g[i-1][j-1];
        }
    int x1,x2,y1,y2;
    while(q--)
    {
        cin >> x1 >> y1 >> x2 >> y2;
        cout << g[x2][y2]-g[x1-1][y2]-g[x2][y1-1]+g[x1-1][y1-1] << endl;
    }
    return 0;
}
```
